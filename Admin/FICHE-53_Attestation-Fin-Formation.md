## Objectifs de la formation

Les compétences visées par cette formation sont les suivantes :

| Code | Compétence |
|:----:|------------|
| **C1** | Comprendre les fondamentaux de l'intelligence artificielle, du machine learning et des LLM (Large Language Models) |
| **C2** | Maîtriser les techniques de prompting avec la méthode CREA pour obtenir des résultats professionnels |
| **C3** | Utiliser les principaux outils d'IA générative (ChatGPT, Claude, Gemini) de manière efficace et raisonnée |
| **C4** | Concevoir des workflows hybrides IA + bureautique adaptés à son métier tertiaire |
| **C5** | Automatiser des tâches répétitives avec un outil no-code (Make) intégrant des modules IA |
| **C6** | Piloter un projet d'intégration de l'IA dans son environnement professionnel |

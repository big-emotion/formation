# Formation IA pour les Métiers du Tertiaire

Dispositif de formation clé en main de **35 heures**, structuré en 10 sessions, destiné aux professionnels du secteur tertiaire (RH, Marketing, Finance, Commercial, Gestion).

> État actuel : **v1.0 en cours de validation** — programme pédagogique en review par les co-fondateurs.

---

## Structure du programme

| Dossier | Contenu | Fiches |
|---|---|---|
| `Module-1_Agents-IA-LLM/` | Fondamentaux : LLM, prompting, RAG, assistants IA, workflows | FICHE-01 → 20 · SLIDES-01 → 05 |
| `Module-2_Pratique-Metier/` | Applications métier : RH, Marketing, Finance, Commercial, Gestion, M365, no-code | FICHE-22 → 45 · SLIDES-06 → 10 |
| `Evaluations/` | QCMs, grilles d'évaluation, fiche projet final, auto-évaluation | FICHE-03, 12, 21, 39, 46 → 49 |
| `Admin/` | Plan d'action post-formation, ressources, satisfaction, attestation | FICHE-50 → 53 |

---

## Comment contribuer

1. Prendre une issue GitHub (une par fiche/deck)
2. Créer une branche `revue/fiche-NN`
3. Ouvrir une PR avec le template — remplir les checklists
4. 2 approvals requis (CODEOWNERS)
5. Merger → mettre à jour le statut dans Notion

→ Détails complets dans [CONTRIBUTING.md](./CONTRIBUTING.md)

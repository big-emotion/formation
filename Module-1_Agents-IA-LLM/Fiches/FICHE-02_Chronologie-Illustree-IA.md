## De 1950 à aujourd'hui : les grandes étapes de l'IA

```
1950 ──── Alan TURING publie "Computing Machinery and Intelligence"
│         → Pose LA question : "Les machines peuvent-elles penser ?"
│         → Invente le "Test de Turing"
│
1956 ──── Conférence de DARTMOUTH
│         → Le terme "Intelligence Artificielle" est inventé
│         → Naissance officielle de la discipline
│
1958 ──── Le PERCEPTRON (Frank Rosenblatt)
│         → Premier réseau de neurones artificiel
│         → Enthousiasme massif... puis déception
│
│  ❄️ PREMIER HIVER DE L'IA (1974-1980)
│  Financement coupé, recherche au ralenti
│
1980 ──── Les SYSTÈMES EXPERTS
│         → Des programmes IA basés sur des règles métier
│         → Utilisés en médecine, finance, industrie
│
│  ❄️ DEUXIÈME HIVER DE L'IA (1987-1993)
│  Les systèmes experts montrent leurs limites
│
1997 ──── DEEP BLUE (IBM) bat Kasparov aux échecs
│         → L'IA bat un champion du monde pour la première fois
│         → Mais c'est de la force brute, pas de l'"intelligence"
│
2011 ──── WATSON (IBM) gagne à Jeopardy!
│         → L'IA comprend le langage naturel (questions en anglais)
│
2012 ──── ⭐ RÉVOLUTION DU DEEP LEARNING
│         → AlexNet gagne ImageNet (reconnaissance d'images)
│         → Les réseaux de neurones profonds prouvent leur efficacité
│         → Grâce à : plus de données + GPU puissants
│
2014 ──── Les GAN (Generative Adversarial Networks)
│         → L'IA apprend à créer des images réalistes
│         → Début de l'IA générative
│
2016 ──── ALPHAGO (Google DeepMind) bat le champion de Go
│         → Le Go était considéré comme "impossible" pour l'IA
│         → Preuve que le deep learning surpasse l'humain sur des tâches complexes
│
2017 ──── ⭐ L'architecture TRANSFORMER est inventée (Google)
│         → Article "Attention Is All You Need"
│         → La fondation technique de TOUS les LLM actuels
│
2018 ──── GPT-1 (OpenAI) + BERT (Google)
│         → Premiers grands modèles de langage
│         → Début de la compréhension du langage à grande échelle
│
2020 ──── GPT-3 (OpenAI)
│         → 175 milliards de paramètres
│         → Peut écrire des articles, du code, des poèmes
│         → Accessible uniquement via API
│
2022 ──── ⭐⭐⭐ CHATGPT (OpenAI) — 30 novembre
│         → GPT-3.5 accessible à TOUS via une interface web
│         → 100 millions d'utilisateurs en 2 mois
│         → LE moment qui change tout pour le grand public
│
│         DALL-E 2, Midjourney, Stable Diffusion
│         → L'IA génère des images à partir de texte
│
2023 ──── GPT-4 (OpenAI) — Mars
│         → Multimodal (texte + image)
│         → Performance proche de l'humain sur de nombreux tests
│
│         Claude (Anthropic), Gemini (Google), Mistral (France)
│         → La concurrence s'intensifie
│         → Mistral AI lève 400M€ — champion français
│
│         AI Act européen adopté
│         → Premier cadre légal mondial sur l'IA
│
2024 ──── GPT-4o, Claude 3.5 Sonnet, Gemini 1.5 Pro
│         → Les modèles deviennent multimodaux (texte, image, audio, vidéo)
│         → Les "agents IA" émergent
│         → L'IA s'intègre dans les outils de bureau (Copilot, Gemini)
│
2025 ──── L'ère des AGENTS IA
│         → Les LLM ne font plus que parler : ils agissent
│         → Intégration massive dans les entreprises
│         → Automatisation no-code accessible à tous
│         → Claude 4, GPT-5, Gemini 2 repoussent les limites
│
VOUS ─── ÊTES ICI 📍
```

## Les 3 grandes ères de l'IA

| Ère | Période | Caractéristique | Exemple |
|-----|---------|----------------|---------|
| **IA symbolique** | 1956-2010 | Règles écrites par des humains | Systèmes experts, Deep Blue |
| **IA par apprentissage** | 2012-2021 | L'IA apprend à partir de données | Reconnaissance d'images, traduction |
| **IA générative** | 2022-présent | L'IA crée du contenu nouveau | ChatGPT, Claude, DALL-E, Midjourney |

---

## Chiffre clé

> **En 2 ans** (2022-2024), l'IA générative est passée de 0 à **+ de 500 millions d'utilisateurs** dans le monde. C'est l'adoption technologique la plus rapide de l'histoire humaine.

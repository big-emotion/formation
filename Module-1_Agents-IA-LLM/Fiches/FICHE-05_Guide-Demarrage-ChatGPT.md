# FICHE-05 — Guide de Démarrage Rapide : ChatGPT

> **Formation IA pour les Métiers du Tertiaire**
> Guide pas-à-pas avec captures d'écran (2 pages)

---

## Étape 1 : Créer un compte

1. Ouvrez votre navigateur et allez sur **chat.openai.com**
2. Cliquez sur **"Sign up"** (S'inscrire)
3. Choisissez votre méthode d'inscription :
   - Email + mot de passe
   - Connexion via Google (recommandé — plus rapide)
   - Connexion via Microsoft
   - Connexion via Apple
4. Confirmez votre email si nécessaire
5. Renseignez votre nom et votre date de naissance

> **Note :** La version gratuite donne accès à GPT-4o mini et un accès limité à GPT-4o. La version Plus (20€/mois) donne un accès complet à GPT-4o, DALL-E, Code Interpreter et les GPTs personnalisés.

| Élément | Fonction |
|---------|----------|

## Étape 3 : Votre première conversation

### Essayez ces prompts pour vous familiariser :

**Prompt 1 — Test simple :**
> Bonjour ! Peux-tu te présenter en 3 phrases ?

**Prompt 2 — Test professionnel :**
> Je suis responsable marketing dans une PME. Rédige un email de 100 mots pour annoncer à nos clients le lancement de notre nouveau service de livraison express.

**Prompt 3 — Test d'analyse :**
> Voici les chiffres de vente du trimestre : Janvier 45 000€, Février 52 000€, Mars 38 000€. Analyse ces chiffres et identifie la tendance.

## Étape 4 : Les fonctionnalités utiles

### Uploader un fichier
1. Cliquez sur l'icône **📎** (trombone) dans la barre de message
2. Sélectionnez votre fichier (PDF, image, CSV, DOCX, XLSX...)
3. Le fichier apparaît dans la conversation
4. Posez votre question sur le fichier

**Exemple :** Uploadez un PDF et demandez "Résume ce document en 5 points clés"

### Régénérer une réponse
- Si la réponse ne vous convient pas, cliquez sur **🔄 Régénérer** sous la réponse
- L'IA produira une nouvelle réponse différente

### Modifier votre prompt
- Cliquez sur l'icône **✏️** à côté de votre message pour le modifier
- ChatGPT régénèrera une réponse basée sur votre message modifié

### Copier une réponse
- Cliquez sur l'icône **📋** (copier) sous la réponse
- Collez dans votre document Google Docs, email, etc.

## Étape 5 : Les GPTs personnalisés (version Plus)

Les GPTs sont des versions de ChatGPT pré-configurées pour des tâches spécifiques.

### Accéder aux GPTs :
1. Cliquez sur **"Explorer les GPTs"** dans la barre latérale
2. Parcourez les catégories : Productivité, Rédaction, Analyse...
3. Cliquez sur un GPT pour l'utiliser

### Créer votre propre GPT :
1. Cliquez sur **"Créer un GPT"**
2. Décrivez ce que vous voulez en langage naturel
3. Ajoutez des documents de référence si nécessaire
4. Testez et affinez
5. Enregistrez (privé ou partagé)

## Raccourcis et astuces

| Raccourci / Astuce | Action |
|--------------------|--------|
| `Entrée` | Envoyer le message |
| `Shift + Entrée` | Aller à la ligne sans envoyer |
| Commencer par `/` | Accéder aux raccourcis |
| "Continue" | Si la réponse est coupée, l'IA reprend où elle s'est arrêtée |
| "En français" | Ajouter à votre prompt si l'IA répond en anglais |

---

## Précautions importantes

- **Ne saisissez JAMAIS** de données personnelles de clients/salariés (noms, emails, n° de sécu...)
- **Ne saisissez JAMAIS** de données confidentielles de l'entreprise (résultats non publiés, secrets commerciaux)
- **Vérifiez toujours** les informations factuelles (dates, chiffres, noms)
- **Ne faites pas confiance** aux sources citées — elles peuvent être inventées
- Les conversations sont utilisées par défaut pour l'entraînement — désactivez dans Paramètres > Contrôle des données si nécessaire

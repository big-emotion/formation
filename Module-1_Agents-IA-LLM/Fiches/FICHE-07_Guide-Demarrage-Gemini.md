# FICHE-07 — Guide de Démarrage Rapide : Gemini

> **Formation IA pour les Métiers du Tertiaire**
> Guide pas-à-pas avec captures d'écran (2 pages)

---

## Étape 1 : Accéder à Gemini

1. Ouvrez votre navigateur et allez sur **gemini.google.com**
2. Connectez-vous avec votre **compte Google** (Gmail)
   - Si vous avez un compte Google Workspace (entreprise), utilisez-le
   - Si non, votre compte Gmail personnel fonctionne
3. Acceptez les conditions d'utilisation
4. Vous êtes prêt — pas besoin de créer un nouveau compte !

> **Note :** Gemini est gratuit avec un compte Google. La version Advanced (21,99€/mois, incluse dans Google One AI Premium) donne accès à Gemini 1.5 Pro, une fenêtre de contexte d'1M de tokens, et l'intégration complète avec Google Workspace.

## Étape 3 : Votre première conversation

### Essayez ces prompts :

**Prompt 1 — Test simple :**
> Bonjour Gemini ! Explique-moi en 3 phrases simples ce que tu sais faire.

**Prompt 2 — Test professionnel :**
> Je suis contrôleur de gestion. Notre CA a augmenté de 12% ce trimestre mais notre marge brute a baissé de 3 points. Aide-moi à formuler 3 hypothèses pour expliquer cet écart au prochain CODIR.

**Prompt 3 — Test avec image :**
> [Uploadez une photo d'un graphique ou tableau] Analyse ce graphique et résume les tendances principales.

## Étape 4 : La force de Gemini — L'intégration Google Workspace

### Gemini dans Google Docs
- Ouvrez un Google Doc
- Cliquez sur **"Aidez-moi à écrire"** (icône stylo + étoile ✨)
- Saisissez votre instruction : "Rédige une note de service sur le télétravail"
- Gemini génère le texte directement dans le document
- Vous pouvez aussi **sélectionner du texte** → clic droit → "Demander à Gemini" pour résumer, reformuler, allonger

### Gemini dans Google Sheets
- Ouvrez un Google Sheets
- Cliquez sur l'icône **Gemini** (✨) dans la barre latérale
- Posez des questions en langage naturel sur vos données :
  - "Quelle est la moyenne de la colonne B ?"
  - "Crée un graphique des ventes par mois"
  - "Quelle formule pour calculer la marge ?"
- Gemini peut aussi **générer des formules** complexes

### Gemini dans Google Slides
- Ouvrez Google Slides
- Cliquez sur **"Aidez-moi à créer"** (✨)
- Décrivez votre présentation : "Crée une présentation de 5 slides sur notre stratégie RSE"
- Gemini génère un squelette de présentation
- Vous pouvez aussi générer des **images** pour illustrer vos slides

### Gemini dans Gmail
- Ouvrez Gmail
- En rédigeant un email, cliquez sur **"Aidez-moi à écrire"** (✨)
- Décrivez l'email souhaité : "Réponds poliment que je ne suis pas disponible cette semaine"
- Gemini rédige l'email dans le champ de composition
- Fonctionne aussi pour **résumer des fils de discussion** longs

> **Important :** Ces fonctionnalités Workspace nécessitent soit Google Workspace avec Gemini activé par l'admin, soit un abonnement Google One AI Premium personnel.

## Étape 5 : Les Gems (assistants personnalisés)

Les Gems sont l'équivalent Gemini des GPTs de ChatGPT.

### Utiliser un Gem existant :
1. Cliquez sur **"Gems"** dans la barre latérale
2. Parcourez les Gems disponibles (rédacteur, analyste, brainstormer...)
3. Sélectionnez un Gem pour démarrer une conversation

### Créer votre Gem :
1. Cliquez sur **"Gems"** → **"Créer un Gem"**
2. Donnez-lui des instructions (équivalent des instructions système)
3. Testez et affinez
4. Sauvegardez

## Astuces Gemini

| Astuce | Détail |
|--------|--------|
| **Double vérification** | Gemini affiche parfois un bouton "Vérifier" — il cherche sur Google si ses réponses sont exactes |
| **Sources** | Gemini peut afficher des liens vers ses sources Google |
| **Multimodal** | Uploadez des images, des PDF, même des vidéos YouTube (collez le lien) |
| **Extensions** | Activez les extensions Google (Maps, YouTube, Flights...) dans les paramètres |
| **Voix** | Utilisez le micro pour dicter vos prompts |

---

## Précautions importantes

- **Compte professionnel vs personnel** : Si vous utilisez un compte Google Workspace d'entreprise, votre admin contrôle les paramètres de confidentialité. Vérifiez avec votre DSI.
- **Ne saisissez pas** de données confidentielles dans la version grand public
- **Vérifiez** les réponses — Gemini peut halluciner comme tout LLM
- **Disponibilité** : Certaines fonctionnalités Workspace ne sont pas encore disponibles dans tous les pays/langues
- Les réponses de Gemini peuvent varier selon que vous utilisez la version gratuite ou Advanced

## Quand utiliser Gemini plutôt que ChatGPT ou Claude ?

| Situation | Pourquoi Gemini |
|-----------|----------------|
| Vous travaillez dans Google Docs/Sheets/Slides | Intégration native, pas de copier-coller |
| Vous avez besoin d'infos à jour | Accès direct à la recherche Google |
| Vous analysez des vidéos YouTube | Collez le lien, Gemini analyse le contenu |
| Votre entreprise est sur Google Workspace | Cohérence de l'écosystème |
| Vous voulez vérifier les sources | Fonction "Vérifier" avec sources Google |

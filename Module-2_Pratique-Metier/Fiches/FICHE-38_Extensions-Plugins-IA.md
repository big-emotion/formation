### Comment installer une extension Chrome

1. Ouvrez le **Chrome Web Store** (chrome.google.com/webstore)
2. Recherchez le nom de l'extension
3. Cliquez sur **Ajouter à Chrome**
4. Vérifiez les permissions demandées avant de confirmer
5. L'icone apparaît dans la barre d'extensions (puzzle en haut à droite)

## 4. Précautions et sécurité

### 4.1 Avant d'installer une extension

Vérifiez systématiquement les points suivants :

| Point de vérification | Détail |
|----------------------|--------|
| **Permissions demandées** | Refusez les extensions qui demandent des accès excessifs (ex : "Lire et modifier toutes vos données sur tous les sites" sans justification) |
| **Nombre d'avis** | Privilégiez les extensions avec plus de 1 000 avis et une note supérieure a 4/5 |
| **Éditeur identifié** | Vérifiez que l'éditeur est clairement identifié et dispose d'un site web |
| **Dernière mise à jour** | Évitez les extensions non mises à jour depuis plus de 6 mois |
| **Politique de confidentialité** | Lisez la politique de confidentialité, surtout pour les extensions qui traitent des données |

### 4.2 RGPD et données confidentielles

> **Règle d'or** : Toute extension qui traite vos données les envoie potentiellement sur des serveurs externes. Soyez vigilant.

| Niveau de risque | Type de données | Recommandation |
|-----------------|----------------|----------------|
| **Faible** | Texte public (articles, pages web) | Utilisation libre avec les extensions |
| **Moyen** | Emails professionnels, documents internes | Vérifiez la politique de confidentialité de l'extension |
| **Élevé** | Données clients, données financières, données RH | N'utilisez que les outils approuvés par votre DSI |
| **Critique** | Données médicales, juridiques, mots de passe | Ne jamais traiter avec des extensions tierces |

### 4.3 Bonnes pratiques en entreprise

1. **Demandez l'avis de votre DSI** avant d'installer des extensions sur un poste professionnel
2. **N'installez pas d'extensions inconnues** recommandées par des sources non vérifiées
3. **Faites un audit régulier** de vos extensions installées (Chrome > Extensions)
4. **Désactivez les extensions inutilisées** plutôt que de les laisser actives
5. **Signalez** toute extension suspecte à votre service informatique

### Slide 8 — Gemini dans Google Docs : generation de texte

## Generer du contenu directement dans Docs

**Comment faire :**
1. Ouvrir un Google Doc
2. Taper `/` ou cliquer sur l'icone Gemini (panneau lateral)
3. Decrire ce que vous voulez generer

**Exemples de prompts dans Docs :**

| Besoin | Prompt dans Gemini |
|--------|-------------------|
| Creer un compte rendu | "Redige un compte rendu de reunion sur le lancement du projet X avec 5 decisions prises" |
| Generer une procedure | "Ecris une procedure d'accueil des nouveaux collaborateurs en 10 etapes" |
| Creer un modele d'email | "Propose 3 versions d'un email de relance client professionnel" |

Le texte est insere **directement dans votre document**.

### Slide 10 — Gemini dans Docs : resume de documents

## Resumer un long document en quelques clics

**Cas d'usage concrets :**
- Resumer un rapport de 20 pages en 1 page
- Extraire les points cles d'un document juridique
- Synthetiser les decisions d'un compte rendu

**Methode :**
1. Ouvrir le panneau lateral Gemini
2. Demander : "Resume ce document en 5 points cles"
3. Affiner : "Quelles sont les actions a mener ?"

**Limites a connaitre :**
- Fonctionne mieux sur des documents < 100 pages
- Ne remplace pas une lecture attentive pour les documents critiques
- Toujours verifier les chiffres et les noms

### Slide 12 — Gemini dans Google Sheets : formules en langage naturel

## Fini les formules complexes a memoriser

**Le probleme classique :**
```
=SI(ET(B2>1000;C2="Valide");B2*0.9;SI(B2>500;B2*0.95;B2))
```
Qui se souvient de cette syntaxe ?

**Avec Gemini :**
> "Calcule une remise de 10% si le montant depasse 1000 et que le statut est Valide,
> 5% si le montant depasse 500, sinon pas de remise"

Gemini genere la formule et l'insere dans la cellule.

**Autres exemples :**
- "Calcule la moyenne des ventes du trimestre 3"
- "Trouve les doublons dans la colonne A"
- "Cree une colonne qui classe les clients par tranche de CA"

### Slide 13 — Gemini dans Sheets : analyse de donnees

## Analyser vos donnees par la conversation

**Dans le panneau lateral Gemini :**

| Votre question | Ce que Gemini fait |
|----------------|-------------------|
| "Quelles sont les tendances dans ces donnees ?" | Analyse statistique + graphique |
| "Quel produit a le plus vendu ce mois ?" | Tri + reponse textuelle |
| "Cree un graphique des ventes par region" | Generation de graphique |
| "Y a-t-il des anomalies dans ces chiffres ?" | Detection d'outliers |

**Bonnes pratiques :**
- Nommez bien vos colonnes (Gemini les utilise pour comprendre)
- Verifiez toujours les formules generees
- Commencez par des questions simples avant les analyses complexes

### Slide 15 — Gemini dans Google Slides : generer des presentations

**Methode :**
1. Ouvrir un nouveau Google Slides
2. Cliquer sur "Creer avec Gemini" ou utiliser le panneau lateral

### Slide 29 — Pourquoi des workflows hybrides ?

## Les limites des IA integrees

```
CE QUE GEMINI/COPILOT FONT BIEN         CE QU'ILS FONT MOINS BIEN
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~          ~~~~~~~~~~~~~~~~~~~~~~~~~~
- Actions rapides dans l'outil           - Prompts tres complexes
- Resume, reformulation simple           - Analyse multi-documents
- Formules et graphiques                 - Persona / ton tres specifique
- Brouillons d'emails                    - Chaines de prompts
                                         - Comparaison de modeles IA
```

**Solution** : utiliser Claude, ChatGPT ou d'autres IA pour les taches complexes, puis integrer le resultat dans vos outils bureautiques.
